# 更新日志

本页面记录 SkiHide 的版本更新内容，包括新功能、改进与修复。  
如未特别说明，所有版本均适用于 **Windows 10 / 11（x64）**。

## v1.3.5（当前版本）

**发布日期：2026-01-02**

**构建号：build.26001**

### ✨ 新增

- 新增 **问题反馈** 入口，如遇问题可直接进入反馈页面

### 📦 其他

- 软件名从 **WindowHide** 正式更名为 **SkiHide**
- 修改软件图标

## v1.3.4

**发布日期：2025-12-14**

**构建号：build.25004**

### ✨ 新增

- 新增 DeBug 模式，启动时添加-DEBUG 即可进入 DeBug 模式

### ⚙️ 改进

- 检查更新逻辑更改

## v1.3.3

**发布日期：2025-12-02**

### ✨ 新增

- 新增 设置页面
- 新增 隐藏后关闭声音功能，默认关闭需在设置页面开启

## v1.3.2

**发布日期：2025-12-01**

### ✨ 新增

- 新增 快捷键等设置的配置保存功能，以便下次打开软件还能使用上次的软件配置

### 🐛 修复

- 修复 托盘程序同时存在 Quit 和退出两个按钮

## v1.3.1

**发布日期：2025-11-17**

### 🐛 修复

- 修复 不可见窗口的列表可见性
- 修复 在 1.3.0 中的更新脚本报错问题

### ⚙️ 改进

- 重构整体代码逻辑，优化整体运行速度

## v1.3.0

**发布日期：2025-10-10**

### ✨ 新增

- 新增 在线更新功能

### ⚙️ 改进

- 移除窗口列表中无用的窗口进程

## v1.0.0（初始版本）

**发布日期：2024-06-21**

### ✨ 新增

- 基本窗口隐藏功能：一键隐藏/恢复指定窗口
- 自定义快捷键设置
- 选择需要隐藏的应用
- 最小化到系统托盘

## 更新方式

- 程序启动后会自动检查新版本
- 若发现更新，会弹窗提示并展示更新内容
- 是否更新完全由用户自行决定
