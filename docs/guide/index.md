# 快速开始

**SkiHide** 是一款 Windows 工具，  
通过 **全局快捷键或鼠标侧键**，  
快速 **隐藏 / 恢复任意窗口**，  
非常适合：

- 🎮 游戏 + 攻略
- 🖥️ 多屏办公
- 📺 演示 / 直播 / 临时整理桌面

## 三步开始使用

### 1.启动程序

解压后直接运行 `SkiHide.exe`，无需安装。

启动后程序会打开**主界面**，点击最小化后软件会常驻后台，图标位于 **系统托盘**。

### 2.设置快捷键或鼠标侧键

- 点击输入框，按下你想要的组合键  
  例如：`Ctrl + Alt + H`
- 或勾选 **“使用鼠标侧键”**

### 3️.试一次

1. 在窗口列表中选中一个窗口
2. 按下快捷键 / 鼠标侧键

窗口会立刻隐藏，再按一次即可恢复。

🎉 完成！
