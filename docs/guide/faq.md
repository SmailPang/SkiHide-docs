# 常见问题

## 快捷键无效？

- 请确认已点击“开始监听”
- 检查是否被其他软件占用
- 全屏游戏请使用窗口化全屏或窗口化

## 窗口不在列表里？

- 系统窗口已被自动过滤
- 工具类窗口默认不显示

## 我的游戏账号被封禁了怎么办？

- 本软件不存在修改游戏内存、注入进程、干预游戏逻辑等行为
- 具体请看 [游戏与第三方平台风险说明](/guide/privacy#游戏与第三方平台风险说明) 和 [用户责任声明](/guide/privacy#用户责任声明)

## 杀毒软件提示？

- SkiHide 不注入、不联网（除更新）
- 可将程序加入信任列表

## 程序崩溃怎么办？

- 查看目录下的 `log.txt`
- 将日志提交到 [反馈页面](https://github.com/SmailPang/SkiHide/issues)

### 注意：提交反馈前请确保你知道应该怎么正确跟开发者反馈问题
